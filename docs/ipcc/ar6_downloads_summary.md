## File Type Definitions

- **Raw HTML (`gatsby_raw.html`, `wordpress_raw.html`):** Downloaded directly from IPCC website, includes navigation, tooltips, etc.
- **Partially Processed (`de_gatsby.html`, `de_wordpress.html`):** Cleaned of navigation/tooltips, but no semantic IDs yet
- **Processed (`gatsby.html`, `wordpress.html`):** Intermediate processing step
- **With IDs (`html_with_ids.html`):** Final output with semantic IDs for paragraphs and sections
- **ID Lists (`id_list.html`):** List of all IDs in the document
- **Para Lists (`para_list.html`):** List of all paragraphs with their IDs
