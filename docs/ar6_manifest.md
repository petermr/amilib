## Processing Pipeline Stages

1. **Scrape** - Download from IPCC website (HTML or PDF)
2. **Convert from PDF** - Extract PDF content and unwrap HTML (if PDF-only)
3. **Convert PDF to Raw HTML** - Transform PDF content into raw HTML pages
4. **Edit HTML to Running HTML** - Clean navigation, tooltips, remove unnecessary markup
5. **Semantify HTML** - Add semantic IDs to paragraphs, sections, and elements
